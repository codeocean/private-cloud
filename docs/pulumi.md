# Pulumi Setup

## Amazon Linux 2

Install Pulumi:
```
curl -fsSL https://get.pulumi.com | sh
source ~/.bashrc
```
Install Node.js:
```
curl -sL https://rpm.nodesource.com/setup_14.x | sudo bash -
sudo yum install -y nodejs
```
Install Git:
```
sudo yum install -y git
```

## macOS

Install Pulumi:
```
brew install pulumi
```
Install Node.js:
```
brew install node@14
```
